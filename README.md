## Development

This site is built with [Astro](https://astro.build). Run all commands from the root of the project:

| Command | Action |
| :--- | :--- |
| `npm install` | Install dependencies |
| `npm run dev` | Start local dev server at `localhost:4321` |
| `npm run build` | Build production site to `./dist/` |
| `npm run preview` | Preview the build locally before deploying |

## Contributing

Contributions are welcome! To add a new resource:

1. Find the relevant file in `src/content/resources/`
2. Add an entry with `title`, `description`, `url`, and `tag`:
   - `Official` — published by Microsoft
   - `Community` — built by the SharePoint/Microsoft 365 community
   - `Free` — created by an individual and free to use (include creator credit in the description)
3. Open a pull request

Every link shared helps someone in the community. **Sharing is Caring.**
